# 🎬 Streaming Platform

منصة عصرية وحديثة لمشاهدة الأفلام والمسلسلات

## ✨ الميزات الرئيسية

- 🎥 مكتبة أفلام ومسلسلات ضخمة
- 🔐 نظام تسجيل دخول آمن (بريد إلكتروني / رقم هاتف)
- 👥 دعوة الأصدقاء والمشاركة
- ⏰ تتبع آخر المشاهدات
- 🎨 تصميم عصري (أسود وأحمر)
- 📱 واجهة responsive على جميع الأجهزة
- 🔔 إشعارات بالحلقات الجديدة
- ⭐ تقييم الأفلام والمسلسلات

## 🛠️ التكنولوجيا المستخدمة

### Frontend
- React.js
- Tailwind CSS
- Redux للإدارة
- Axios للـ API

### Backend
- Node.js & Express
- MongoDB لقاعدة البيانات
- JWT للمصادقة
- Socket.io للإشعارات الفورية

## 📂 هيكل المشروع
